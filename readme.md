### gofm

a simple tui file manager written in golang

inspired by [yazi](https://github.com/sxyazi/yazi)

tui widget libary used - [tview](https://github.com/rivo/tview)

features
- navigate folders with vim keys (hjkl)
- open files via `xdg-open`
- cut/copy/paste
- delete files (confirm dialog box)
- preview pane (for text files and folders)
